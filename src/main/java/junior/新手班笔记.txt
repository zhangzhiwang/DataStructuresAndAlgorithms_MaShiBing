二、算法
1、什么是算法
算法就是解决一个问题的流程设计。
（1）有一个待解决的问题
（2）设计解决这个问题的具体流程
（3）有一个可量化的评价体系来评测流程设计得怎么样
2、算法离不开数据结构
3、算法其实就两种：一种是明确知道怎么算，一种是知道怎么试。
（1）计算器就属于前者，要想算一个数就必须明确知道每一步该怎么算才能使用计算器
（2）计算机就属于后者，要想算出德军情报的密码，我不知道每一步该怎么把它算出来，但是我知道怎么把最终结果给试出来，这就是计算机，所以计算机科学是科学。

三、数据结构
1、什么是数据结构
数据结构是存储和组织数据的方式。选择合适的数据结构可以带来更高的运行或者存储效率。数据结构是算法的载体。
2、数据结构的分类
数据结构一共分量两种：一种是连续结构，一种是跳转结构。连续结构就是内存是连续的，跳转结构内存不是连续的。
3、数组是常见的连续型数据结构，它便于查找，不便于增删数据，因为要维持内存连续；链表是常见的跳转型数据结构，它便于增删数据，不便于查找，因为内存是不连续的，不知道某一个元素在内存的哪个位置。
4、任何语言都有随机函数，在java中就是Math.random()
5、对数器
对数器就是验证算法的正确性，用两种方式来求解，如果在大概率样本的测试中两种算法的结果一致，那么被验证的目标算法就是正确的。另一种用来验证的实现只要保证正确性即可，不需要保证时间和空间复杂度。

四、单链表
1、什么是单链表和双链表
  链表属于跳转型数据结构，非连续内存，所以某一个节点必须记录下一个节点的内存地址，这样就组成了链表。
  （1）单链表
  每个节点除了保存值以外还要保存下一个节点的内存地址
  class Node {
      Object value;// 保存值
      Node nect;// 保存下一个节点的引用
  }
  （2）双链表
  每个节点除了保存值以外还要保存下一个节点和上一个节点的内存地址
  class Node {
      Object value;// 保存值
      Node nect;// 保存下一个节点的引用
      Node last;// 保存上一个节点的引用
  }

3、栈
